# Jogo-Silencio-Inoxidavel

SOBRE O PROJETO:

Silêncio Inoxidável é um jogo de terror e sobrevivência desenvolvido em Python utilizando a biblioteca Pygame como projeto final da disciplina de Informática Básica. 

O jogador assume o papel de um invasor que entra na antiga Escola Técnica de Tecnologia Avançada (ETTA), um local abandonado após um misterioso acidente envolvendo inteligência artificial. Preso dentro do prédio e perseguido por um android infectado, o jogador deve explorar os corredores da escola, encontrar componentes eletrônicos e montar um robô aliado capaz de hackear o sistema de segurança e abrir as portas para a fuga.

COMO EXECUTAR: 

1. Instale o Python;
Baixe e instale a versão mais recente do Python.

2. Instale o Pygame;
pip install pygame

4. Instale o opencv: 
pip install opencv-python

5. Instale o numpy:
pip install numpy

6. Execute o jogo;
python silencio_inoxidavel.py


CONTROLES:

Digite as opções exibidas na tela para tomar decisões.
Pressione Enter para confirmar a escolha.
Feche a janela para encerrar o jogo.
